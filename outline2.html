How to use this board

  • 1) Choose a Type (Act, Beat, Scene, Character, Note), add a Title and optional Content, then “Add Card”.
  • 2) Acts create full-width dividers and separate rows. Beats and Scenes snap into a uniform grid.
  • 3) Drag cards to reorder; the dotted box shows where they’ll snap.
  • 4) Start a content line with “=” to mark arcs/plot points as chips.
  • 5) Use @Tags anywhere (e.g., @B-Story, @Kayla). Tags show as chips and can be filtered above.
  • 6) Import beatsheets (.fountain): “#” becomes Act, “##” Beat, “###” Scene. Export to .fountain anytime.
  • 7) Save/Open JSON files to keep multiple projects. “New / Clear Board” resets the board.
